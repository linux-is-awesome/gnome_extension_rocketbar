## Manual Installation

Stable Version
```
curl -L https://github.com/linux-is-awesome/gnome_extension_rocketbar/archive/refs/heads/master.zip -o gnome_extension_rocketbar.zip && unzip -q -o gnome_extension_rocketbar.zip && (cd gnome_extension_rocketbar-master/scripts && ./install) && rm -rf gnome_extension_rocketbar-master gnome_extension_rocketbar.zip
```

Unstable Version
```
curl -L https://github.com/linux-is-awesome/gnome_extension_rocketbar/archive/refs/heads/develop.zip -o gnome_extension_rocketbar.zip && unzip -q -o gnome_extension_rocketbar.zip && (cd gnome_extension_rocketbar-develop/scripts && ./install) && rm -rf gnome_extension_rocketbar-develop gnome_extension_rocketbar.zip
```
